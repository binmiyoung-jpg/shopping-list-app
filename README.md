# Shopping List App

A simple and elegant shopping list web application.

## Features

- **Add Items**: Add items by typing and pressing Enter or clicking the Add button
- **Check/Uncheck**: Click the checkbox to mark items as completed
- **Delete Items**: Remove individual items with the delete button
- **Clear Completed**: Remove all completed items at once
- **Persistent Storage**: Data is saved in localStorage and persists after page refresh

## Usage

Simply open `shopping-list.html` in any modern web browser.

## Screenshots

The app features a clean, modern UI with:
- Gradient header design
- Smooth animations
- Responsive layout for mobile devices

## Tech Stack

- HTML5
- CSS3 (with animations and gradients)
- Vanilla JavaScript
- localStorage API
